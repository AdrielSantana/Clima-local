# Clima local
------------------------------
### Projeto de estudo para consumição de APIs, nesse caso, uma API que atualiza o clima de acordo com sua localização.  

Link do site: [Clima local](https://adrielsantana.github.io/Clima-local/weather-location/public/)

O site é feito utilizando as tecnologias:

* React / JSX
* HTML5
* CSS3
* BOOTSTRAP
* JAVASCRIPT
